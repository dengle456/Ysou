# YunSou
 一款HTTP API 对接前端C# GUI.

## 对接格式 
{
    "code": "",
    "msg": "",
    "info": {
        "0": {
            "name": "",
            "Otherinfo": "",
            "filesource": "",
            "uuid": "",
            "type": "",
            "user": "",
            "size": "",
            "pass": "",
            "url": "",
            "sharedtime": ""
        }
    }
}
